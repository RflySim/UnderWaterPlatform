# Experiment Data

This directory contains selected CSV logs for SIL/HIL and localization experiments.

| Path | Description |
|---|---|
| `sil_hil/sil/` | Software-in-the-loop logs |
| `sil_hil/hil/` | Hardware-in-the-loop logs |
| `location1/`, `location2/`, `location3/` | Localization experiment runs |
| `Data/sil/`, `Data/hil/` | Legacy copy of SIL/HIL logs retained for compatibility |

See `../docs/DATASETS.md` for file naming conventions, typical channel meanings, and loading examples.
